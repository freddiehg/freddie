# freddie

freddie is a framework for turning events into typed mutations of an application's state. rayban is a library within it (the typed mutable path), and mercury is its first concrete use, a keyboard-remapping application.

## Other realms

Because freddie can change keys, but freddie is multi-talented and versatile, the intent is to make freddie usable in other realms, such as routing, driving the state of a reactive UI, and other input-to-action domains that fit the same accumulate, resolve, and dispatch shape.
